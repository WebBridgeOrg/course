HTML Tag Reference

Reference of most used HTML tags

  • Title: h1, h2
  • Text: link: a, strong, emphasis
  • Image: img
  • Containers: paragraph, inline: span, block: div
  • Lists: unordered: ul, ordered: ol

h1

Section Heading Level 1

The title of this page is an h1 tag

Code

            <h1>HTML Tag Reference</h1>

Result

HTML Tag Reference

h2

Section Heading Level 1, you can go until level 6 (h3, h4, h5, h6)

The title of this section is an h2 tag

Code

            <h2>h2</h2>

Result

h2

p

Paragraph of text

This line and the one before are paragraphs

Code

           <p>Paragraph of text</p>

Result

Paragraph of text

img

Image

An image displaying the WebBridge logo, an alternative text saying "Web Bridge Logo" and a width of 400 pixels:

Code

           <img src="img/WebBridgeStdLogo.png" alt="Web Bridge Logo" width=400>

Result

a

Link (anchor)

Text that when clicked navigates to the provided web address

Code

           <a href="https://webbridgeorg.github.io/">Web Bridge Website</a>

Result

Web Bridge Website

ul

Unordered List

The ul tag wraps cero or more <li> tags (List Item) and shows an unordered list, this means, a list that is not numerated, the items are presented in the order they appear in the HTML document

Code

           <ul>
               <li>HTML: Hypertext Markup Language</li>
               <li>CSS: Cascade Style Sheet</li>
               <li>JS: Javascript</li>
           </ul>

Result

  • HTML: Hypertext Markup Language
  • CSS: Cascade Style Sheet
  • JS: Javascript

ol

Ordered List

The ol tag wraps cero or more <li> tags (List Item) and shows an ordered list, this means, a list that is numerated

Code

           <ol>
               <li>HTML: Hypertext Markup Language</li>
               <li>CSS: Cascade Style Sheet</li>
               <li>JS: Javascript</li>
           </ol>

Result

  1. HTML: Hypertext Markup Language
  2. CSS: Cascade Style Sheet
  3. JS: Javascript

strong

Strong Text

Usually presented as bold text, but not necesarilly, it can be styled with CSS to display in any other way

Code

           <strong>This is important</strong>

Result

This is important

em

Emphasized Text

Usually presented as italic text, but not necesarilly, it can be styled with CSS to display in any other way

Code

           <em>This requires emphasis</em>

Result

This requires emphasis

span

Generic inline container

Used to group elements for styling purposes, or because they share attribute values.

It should be used only when no other semantic element is appropriate

Code

           <span class="grouped-things">Line <em>with</em> different <strong>things</strong></span>

Result

Line with different things

div

Generic block container

Used to group elements for styling purposes, or because they share attribute values.

It should be used only when no other semantic element is appropriate

Code

           <div class="tag-doc">
               <h2>div</h2>
               <p class="tag-short-desc">Generic block container</p>
               <p class="tag-desc">Used to group elements for styling purposes,
               or because they share attribute values.</p>
               <p class="tag-desc">It should be used only when no other semantic
               element is appropriate</p>

               <h3>Code</h3>
               <pre><!-- inception --></pre>

               <h3>Result</h3>
               <p>This whole block is a wrapped in a div</p>
           </div>

Result

This whole block is a wrapped in a div
