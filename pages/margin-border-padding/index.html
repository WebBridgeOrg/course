Margin

Usage

       /* margin: $all-sizes; */
       margin: 1em;

Same as

       /* margin: $top-bottom $left-right; */
       margin: 1em 1em;

Same as

       /* margin: $top $right $bottom $left; */
       margin: 1em 1em 1em 1em;

Same as

       margin-top: 1em;
       margin-right: 1em;
       margin-bottom: 1em;
       margin-left: 1em;

Padding

Usage

       /* padding: $all-sizes; */
       padding: 1em;

Same as

       /* padding: $top-bottom $left-right; */
       padding: 1em 1em;

Same as

       /* padding: $top $right $bottom $left; */
       padding: 1em 1em 1em 1em;

Same as

       padding-top: 1em;
       padding-right: 1em;
       padding-bottom: 1em;
       padding-left: 1em;

Border

Usage

       /* border: $all-sizes $style $color; */
       border: 1em solid #B3EFFF;

Same as

       /* border-width: $all-sizes; */
       border-width: 1em;
       
       border-style: solid;
       border-color: #B3EFFF;

Same as

       /* border-width: $top-bottom $right-left; */
       border-width: 1em 1em;
       
       border-style: solid;
       border-color: #B3EFFF;

Same as

       border-top-width: 1em;
       border-right-width: 1em;
       border-bottom-width: 1em;
       border-left-width: 1em;
       
       border-style: solid;
       border-color: #B3EFFF;
